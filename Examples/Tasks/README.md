# Технические задания для дата-журналистики

Учебный набор из **30 сюжетов** в формате редакционного материала: заголовок с выводом, подзаголовок с контекстом, данные, аналитическая подпись, копирайт и источники. Каждый сюжет — отдельная папка с текстом задания и CSV для построения графика в [редакторе](../../index.html).

Данные **вымышленные или агрегированные для обучения** — не публикуйте как новость без проверки.

## Что лежит в папке

```
Examples/Tasks/
├── README.md                 ← этот файл
├── task-01-ceny-benzin/
│   ├── tz.md                 ← техническое задание (markdown)
│   └── data.csv              ← полные данные для импорта
├── task-02-ocheredi-poliklinik/
│   └── …
└── task-30-sea-level-tide-gauges/
    └── …
```

| Файл | Содержание |
|------|------------|
| **tz.md** | Заголовок, подзаголовок, краткое ТЗ на визуализацию, превью таблицы (первые 18 строк), аналитическая подпись, копирайт, источники, подсказка по разделителям CSV |
| **data.csv** | Все строки данных. В длинных сюжетах — сотни записей и **несколько именованных рядов** (регион, категория, линейка и т.д.) |

Примеры **не используют** пары столбцов X/Y для scatter — только категории, время, география и именованные серии, как в обычной дата-журналистике.

## Как работать с заданием

1. Откройте **`tz.md`** — прочитайте заголовок, подзаголовок и блок «Задание для визуализации» / *Visualization brief*.
2. Откройте **`index.html`** в браузере (корень проекта `freehtml2`).
3. Загрузите **`data.csv`** из папки задания (перетаскивание или выбор файла).
4. В панели данных выставьте **разделитель полей** и **десятичный разделитель** — см. таблицу ниже или блок «Параметры CSV» в `tz.md`.
5. Назначьте столбцы на оси и серии (цвет / группировка), выберите тип диаграммы под сюжет.
6. Оформите текст документа: заголовок и подпись из `tz.md`, источники — из раздела «Источники».

**Подсказки по типам графиков**

| Структура данных | Подходящие типы |
|------------------|-----------------|
| Одна категория — одно значение | Столбцы, горизонтальные столбцы, леденец |
| Период + несколько рядов (длинный CSV) | Линия, группированные столбцы, область |
| Две категории + значение | Тепловая карта |
| Широкая таблица (месяц + столбцы стран) | Несколько рядов через преобразование или импорт длинного формата |

## Настройки CSV в редакторе

В правой панели редактора:

- **Разделитель полей** — `,` (запятая) или `;` (точка с запятой).
- **Дес. разделитель** — `.` (точка) или `,` (запятая).

| Обозначение в таблице | Значение в редакторе |
|-----------------------|----------------------|
| поля `,` | Запятая |
| поля `;` | Точка с запятой |
| дес. `.` | Точка |
| дес. `,` | Запятая |

**Английские задания (21–30):** десятичный разделитель в файлах **всегда точка** — в редакторе выбирайте **дес. `.`**.

Если числа не распознаются, сверьте оба параметра с колонкой «CSV» в каталоге.

## Пересоздание примеров

Исходник данных и текстов — скрипт в корне репозитория:

```bash
cd /path/to/freehtml2
node gen-tasks-examples.mjs
```

Скрипт:

- создаёт или обновляет все 30 папок `task-NN-…`;
- записывает `tz.md` и `data.csv`;
- удаляет устаревшие папки `task-*`, которых нет в списке заданий.

После правки сюжетов в **`gen-tasks-examples.mjs`** запустите команду снова, чтобы файлы в `Examples/Tasks/` совпали с кодом.
