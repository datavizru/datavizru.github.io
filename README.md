# FreeHTML Charts

**Бесплатный редактор диаграмм и макета документа в браузере** — без установки, без серверной части. Загружаете CSV, настраиваете график и типографику, экспортируете PNG/SVG или сохраняете проект в JSON.

Интерфейс на русском. Под капотом — [Vega-Lite 5](https://vega.github.io/vega-lite/) и [Vega-Embed](https://github.com/vega/vega-embed).

---

## Возможности

- **Типы графиков:** столбчатые и линейчатые (в т.ч. с накоплением и нормировкой), линейные и ступенчатые, площадные, точечные, леденцовые (вертикальные и горизонтальные), тепловая карта.
- **Данные:** импорт CSV с выбором разделителя полей (`,`, `;`, таб, `|`) и десятичного разделителя (`.` / `,`); несколько рядов через столбец «серия».
- **Документ:** заголовок, подзаголовок, подписи осей, комментарий, футер (копирайт и источники) — выравнивание в рамке листа.
- **Оформление:** шрифты (Google Fonts + локальные из `Fonts/`), оси, сетка, легенда, подписи значений, палитры, шаблоны стиля.
- **Тепловая карта:** своя градиентная шкала, легенда, привязка стопов к min/max данных.
- **Экспорт:** PNG 2× / 4×, SVG; сохранение и загрузка проекта `.json`.
- **Примеры:** встроенные наборы данных и [30 учебных ТЗ для дата-журналистики](Examples/Tasks/README.md).

## Быстрый старт

### Вариант 1 — открыть файл

Дважды щёлкните `index.html` или откройте его в браузере. Для загрузки CSV через `file://` обычно достаточно современного Chrome / Firefox / Safari.

### Вариант 2 — локальный сервер (рекомендуется)

```bash
git clone https://github.com/YOUR_USERNAME/freehtml2.git
cd freehtml2

# Python 3
python3 -m http.server 8080

# или Node.js
npx --yes serve -p 8080
```

Откройте в браузере: `http://localhost:8080`

### Первые шаги в редакторе

1. **Загрузить CSV…** — выберите файл (например `Examples/prodazhi-po-mesyacam.csv`).
2. Укажите **разделитель полей** и **десятичный разделитель**, если числа не подхватились.
3. Назначьте столбцы на **ось X**, **значение Y** и при необходимости **ряд (серия)**.
4. Выберите **тип диаграммы** и заполните текст документа слева.
5. Справа настройте шрифты, цвета и отступы; при необходимости примените **шаблон оформления**.
6. **Экспорт** — PNG или SVG; **Сохранить** — `chart-project.json`.

## Структура репозитория

```
freehtml2/
├── index.html              # приложение (один HTML-файл + встроенные стили и логика)
├── README.md
├── Examples/               # демо-CSV
│   └── Tasks/              # 30 ТЗ (tz.md + data.csv) для журналистики
├── CSV/                    # дополнительные примеры таблиц
├── templates/              # JSON-шаблоны оформления
├── Fonts/                  # локальные шрифты + fonts-manifest.js
├── gen-examples-csv.mjs    # генерация CSV в Examples/
├── gen-tasks-examples.mjs  # генерация Examples/Tasks/
├── gen-fonts-manifest.py   # обновление манифеста шрифтов (без Node)
├── gen-fonts-manifest.mjs  # то же, если установлен Node.js
├── gen-templates-manifest.py   # список шаблонов → templates/manifest.json
└── gen-templates-manifest.mjs  # то же, если установлен Node.js
```

| Папка / файл | Назначение |
|--------------|------------|
| [Examples/](Examples/) | Готовые CSV: месяцы, категории, несколько рядов, теплокарта |
| [Examples/Tasks/](Examples/Tasks/) | Полные технические задания (RU 01–20, EN 21–30) — см. [README](Examples/Tasks/README.md) |
| [templates/](templates/) | Пресеты: светлый, газетный |
| [Fonts/](Fonts/) | Собственные файлы шрифтов (опционально) |

## Типы диаграмм

| Код в приложении | Описание |
|------------------|----------|
| `bar` / `barh` | Столбчатая / линейчатая |
| `barStack` / `barhStack` | С накоплением |
| `barStackNorm` / `barhStackNorm` | Нормированная (100%) |
| `lollipop` / `lollipopH` | Леденцовая (вертикальная / горизонтальная) |
| `line` | Линейный график |
| `lineStepBefore` / `lineStepAfter` | Ступенчатый |
| `area` | Площадная |
| `point` | Точечная |
| `heatmap` | Тепловая карта |

## Учебные задания (Tasks)

В [Examples/Tasks/](Examples/Tasks/) — **30 сюжетов** в формате редакционного материала:

- **01–20** — русский язык, разные разделители CSV;
- **21–30** — English, десятичный разделитель только **точка**.

Каждая папка: `tz.md` (заголовок, данные, подпись, источники) и `data.csv`.

Пересоздать все задания из скрипта:

```bash
node gen-tasks-examples.mjs
```

## Скрипты разработки

Требуется **Node.js 18+** (только для генерации файлов, не для работы редактора).

```bash
# CSV-примеры в Examples/
node gen-examples-csv.mjs

# 30 журналистских заданий
node gen-tasks-examples.mjs

# манифест локальных шрифтов после добавления файлов в Fonts/
python3 gen-fonts-manifest.py
# или: node gen-fonts-manifest.mjs

# список шаблонов в выпадающем меню (все *.json в templates/, кроме manifest.json)
python3 gen-templates-manifest.py
# или: node gen-templates-manifest.mjs
```

## Публикация на GitHub Pages

1. В репозитории: **Settings → Pages → Build from branch**.
2. Branch: `main`, folder: `/ (root)`.
3. Сайт будет доступен по адресу `https://YOUR_USERNAME.github.io/freehtml2/` (откроется `index.html`).

Убедитесь, что в корне лежит `index.html` и относительные пути (`./Fonts/`, `./templates/`) совпадают со структурой репозитория.
